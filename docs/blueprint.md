# **App Name**: VigorStride

## Core Features:

- User Profile: Allow users to create a profile, specifying information such as height, weight, age and preferred units for metrics.
- Workout Logging: Enable users to log different kinds of workout routines, tailored to each kind of activity with fields like reps, sets, weight, distance, and time.
- Goal Setting: Allow users to set their health and activity goals for the day. 
- Progress Visualization: Visually present workout data and goal progression using interactive charts and graphs, displaying trends and progress over time.
- Water Intake Tracking: Track daily water intake with reminders for the user.
- AI Powered Workout Recommendations: Based on the user's logged workouts, use an LLM tool to recommend appropriate next exercises and goals.

## Style Guidelines:

- Primary color: Vibrant green (#90EE90) for motivation and energy.
- Background color: Dark gray (#282828) provides a sophisticated backdrop that emphasizes the vibrant green.
- Accent color: Lime green (#32CD32) used in the application to provide highlights in the interface.
- Body and headline font: 'PT Sans' (sans-serif) to balance readability and a modern touch across all text elements.
- Use clear and dynamic icons to represent different workout types and progress indicators.
- Ensure a clean and intuitive layout with a focus on usability.
- Use subtle animations and transitions to provide feedback and guide users through the app, enhancing the user experience without distraction.
